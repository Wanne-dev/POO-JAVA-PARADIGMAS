# 03 - Herencia

Una clase hereda atributos y métodos de otra.

## Archivos típicos
- `Vehiculo.java` (padre) → Atributos: marca, modelo. Método: encender().
- `Carro.java` (hijo) → Hereda y añade numeroPuertas.
- `Moto.java` (hijo) → Hereda y añade tipoManillar.
- `Main.java` → Crea objetos y usa herencia.

## ¿Qué se aprende?
- `extends`, `super()`, `@Override`.
- Reutilización de código.
