# 01 - Encapsulamiento

Aquí aprendemos a ocultar los datos de un objeto y controlar cómo se accede a ellos.

## Archivos típicos en esta carpeta
- `Persona.java` → Atributos privados (nombre, edad) con getters y setters.
- `CuentaBancaria.java` → Saldo privado, métodos depositar/retirar.
- `Main.java` → Prueba de los objetos.

## ¿Qué se aprende?
- Usar `private` y métodos `public` get/set.
- Proteger la integridad de los datos.
