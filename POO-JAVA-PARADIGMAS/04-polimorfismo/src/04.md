# 🎭 04 - Polimorfismo

Un mismo método se comporta diferente según el objeto.

## Archivos típicos
- `Animal.java` → Método `hacerSonido()`.
- `Perro.java` → Sobrescribe con "Guau".
- `Gato.java` → Sobrescribe con "Miau".
- `Main.java` → Lista de Animales, cada uno hace su sonido.

## ¿Qué se aprende?
- Variable de tipo padre apuntando a objeto hijo.
- Ejecución del método del objeto real.
