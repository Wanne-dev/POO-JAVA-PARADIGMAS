# 🔗 05 - Asociación

Relación "usa un" entre clases independientes.

## Archivos típicos
- `Profesor.java` → Atributos: nombre, lista de Clases.
- `Clase.java` → Atributos: nombre, horario, referencia a Profesor.
- `Main.java` → Asocia profesor con clases.

## ¿Qué se aprende?
- Clases separadas que se conocen.
- No hay dependencia de ciclo de vida.
