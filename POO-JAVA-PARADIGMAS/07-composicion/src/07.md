# 07 - Composición

Relación "tiene un" fuerte: las partes no existen sin el todo.

## Archivos típicos
- `Habitacion.java` → nombre, metros.
- `Casa.java` → crea las habitaciones dentro de sí misma.
- `Main.java` → Al destruir la casa, las habitaciones también.

## ¿Qué se aprende?
- Las partes se crean y destruyen con el todo.
- Mayor acoplamiento pero control total.
